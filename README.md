# Neft va gaz konlari geologiyasi hamda qidiruvi instituti  
## Rasmiy veb-sayt backend xizmati

Ushbu repository **Neft va gaz konlari geologiyasi hamda qidiruvi institutining**
rasmiy veb-sayti uchun backend (server-side) qismini o‘z ichiga oladi.

Backend tizimi institut faoliyatiga oid ma’lumotlarni boshqarish, saqlash
va frontend ilovalariga xavfsiz hamda barqaror API orqali taqdim etish
uchun ishlab chiqilgan.

## Loyiha maqsadi

Mazkur backend tizim quyidagi vazifalarni bajaradi:

- Institut faoliyatiga oid rasmiy ma’lumotlarni markazlashgan holda boshqarish
- Yangiliklar, e’lonlar va ilmiy materiallarni saqlash va tarqatish
- Foydalanuvchi so‘rovlarini API orqali qayta ishlash
- Ma’lumotlar xavfsizligi va yaxlitligini ta’minlash

## Texnologiyalar

- Dasturlash tili: Python
- Backend framework: Django / Django REST Framework
- Ma’lumotlar bazasi: PostgreSQL
- Autentifikatsiya: JWT
- Server muhiti: Linux
- API uslubi: REST
